# @guulabs/react-native-app-badge

App badge for iOS.

![image info](./ios-badge-screenshot.png)

## Installation

```sh
yarn add @guulabs/react-native-app-badge
```

## Usage

```js
import { setAppBadgeCount } from '@guulabs/react-native-app-badge';

// ...

// Set badge
setAppBadgeCount(3)

// Clear badge
setAppBadgeCount(0)
```
